# Plan: 编辑历史列表倒序显示

## 目标
将编辑历史列表改为倒序（最新在最上方），方便用户快速找到最近的操作。

## 修改文件
`desktop/src/components/Editor/EditorToolbar.tsx` 第 107-135 行的 `historyContent`

## 修改内容

将 `history.map((entry, idx) => ...)` 改为倒序遍历：

```tsx
const historyContent = (
  <div style={{ maxHeight: 320, overflowY: 'auto', minWidth: 200 }}>
    {[...history].reverse().map((entry, _) => {
      const idx = history.length - 1 - _;
      return (
        <div
          key={idx}
          onClick={() => onJumpTo?.(idx)}
          style={{
            // ... 样式不变
          }}
        >
          <span ...>{entry.description}</span>
          <span ...>{formatTimeAgo(entry.timestamp)}</span>
        </div>
      );
    })}
  </div>
);
```

## 关键点
- 使用 `[...history].reverse()` 避免修改原数组
- 反向映射时通过 `history.length - 1 - _` 还原真实索引，确保 `onJumpTo(idx)` 和 `idx === currentIndex` 高亮判断正确
