# Plan: 文章版本列表页面 + Diff弹窗

## 需求概述
在文章列表页点击"版本"列时，跳转到一个独立的版本列表页面。每个版本条目显示版本号、时间、查看按钮，点击查看弹窗显示diff差异。

## 现有代码分析
- **后端API已完成**：`GET /api/articles/{id}/versions`（列表）、`GET /api/articles/{id}/versions/{vid}/diff`（差异）
- **前端API service已完成**：`apiService.getArticleVersions()`、`apiService.getVersionDiff()`
- **类型已完成**：`ArticleVersionBrief`、`VersionDiff`
- **现有VersionHistory.tsx**：是Drawer抽屉组件，仅在ArticleEditor编辑器中使用
- 需要新建一个独立的**页面级组件**

## 实现步骤

### 1. 新建版本列表页面 `desktop/src/pages/ArticleVersionList.tsx`
- 新页面组件，使用 `useParams` 获取 `articleId`
- 使用 Ant Design `Table` 组件展示版本列表
- 调用 `apiService.getArticleVersions(articleId)` 获取数据
- 表格列：版本号（version_number）、时间（created_at）、操作（查看按钮）
- 添加面包屑或返回按钮，返回文章列表页
- 点击"查看"按钮打开Modal弹窗

### 2. Diff弹窗
- 在页面内使用 Ant Design `Modal` 组件
- 点击查看时调用 `apiService.getVersionDiff(articleId, versionId)` 获取diff数据
- 复用 `VersionHistory.tsx` 中 `renderDiffContent()` 的diff渲染逻辑（标题变更 + 统一差异格式，颜色区分增/删/改）

### 3. 添加前端路由
- 在 `App.tsx` 中添加路由：`/articles/:id/versions` → `ArticleVersionList`
- 路由需要放在 `articles/:id/edit` 之前以避免匹配冲突

### 4. 修改 ArticleList.tsx
- 将版本列（第130-136行）的 `version_count` 文本改为可点击链接
- 点击时 `navigate(/articles/${record.id}/versions)`

## 文件变更清单
| 文件 | 操作 | 说明 |

| `desktop/src/pages/ArticleVersionList.tsx` | 新建 | 版本列表页面 + Diff弹窗 |
| `desktop/src/App.tsx` | 修改 | 添加 `/articles/:id/versions` 路由 |
| `desktop/src/pages/ArticleList.tsx` | 修改 | 版本列添加点击跳转 |

## UI设计
- 页面顶部：标题"版本历史" + 返回按钮（返回文章列表）
- 主体：Table表格，列为 版本号、变更摘要、创建时间、操作（查看按钮）
- 弹窗：标题显示版本号，内容区显示diff差异（绿色=新增，红色=删除，蓝色=文件头，灰色=hunk头）
