# 媒体库文件存储路径重构计划

## 需求

1. **媒体库上传的文件** → 存放到基础存储路径下的 `media` 文件夹（替代当前的 `uploads/` 目录）
2. **图文编辑中上传/使用媒体库图片** → 复制到对应文章的 `images` 文件夹下
3. **旧数据** → 直接删除，无需迁移

## 现状

| 当前存储类型 | 物理路径 | URL |
|---|---|---|
| 独立上传（媒体库） | `server/uploads/<snow_id>.<ext>` | `GET /uploads/<snow_id>.<ext>` |
| 文章图片 | `server/storage/<uid>/articles/<title>-<id>/images/<file>` | `GET /storage-files/<uid>/articles/.../images/<file>` |

## 目标路径结构

```
server/storage/<user_id>/
  media/                          ← 媒体库文件（新增，替代 uploads/）
    <snow_id>.<ext>
  articles/
    <title>-<snow_id>/
      index.md
      images/                     ← 文章图片（保持不变）
        <timestamp>_<snow_id>.<ext>
  videos/
    ...
```

## 实施步骤

### Step 1: `server/app/routes/settings.py` — 新增 `get_media_dir()`

- 新增函数 `get_media_dir(db, user_id)` 返回 `_get_base_storage_dir(db, user_id) / "media"`
- 自动创建目录

### Step 2: `server/app/config.py` — 移除 `UPLOADS_DIR` 相关

- 删除 `UPLOADS_DIR` 常量
- 从启动时目录创建列表中移除 `UPLOADS_DIR`
- 从导入中移除（在 `media.py` 等文件中也要更新导入）

### Step 3: `server/app/routes/media.py` — 修改 `upload_file`

- 导入 `get_media_dir` 替代 `UPLOADS_DIR`
- 文件保存到 `get_media_dir(db, current_user.id) / <snow_id><ext>`
- `file_path` 记录为相对于 `BASE_STORAGE_DIR` 的路径：`<user_id>/media/<snow_id><ext>`

### Step 4: `server/app/routes/media.py` — 修改 `delete_media_file`

- 移除 `UPLOADS_DIR` 相关逻辑
- 统一使用 `BASE_STORAGE_DIR / media.file_path` 查找文件

### Step 5: `server/app/routes/media.py` — 修改 `copy_media_to_article`

- 源路径解析：`BASE_STORAGE_DIR / media.file_path`（不再需要 `uploads/` 特殊处理）

### Step 6: `server/app/main.py` — 移除 `/uploads/` 端点

- 删除 `serve_uploaded_file` 端点
- 移除 `UPLOADS_DIR` 导入

### Step 7: `desktop/src/services/api.ts` — 更新前端路径解析

- `getMediaUrl()` 方法：移除 `uploads/` 分支
- 新路径格式 `<uid>/media/<file>` 走 `storage-files/` 前缀（因为以数字开头不匹配 `articles/` / `videos/` / `images/`，需添加匹配规则）

### Step 8: `desktop/src/utils/markdown.ts` — 更新 markdown 渲染

- `resolveMediaUrl()` 中添加对 `media/` 路径的处理（或通过数字开头的路径匹配 `storage-files/`）

### Step 9: 清理旧数据

- 删除 `server/uploads/` 目录及其内容
- 清理 DB 中旧媒体记录（file_path 以 `uploads/` 开头的）

## 涉及文件

| 文件 | 变更类型 |
|---|---|
| `server/app/config.py` | 移除 UPLOADS_DIR |
| `server/app/routes/settings.py` | 新增 get_media_dir() |
| `server/app/routes/media.py` | 修改 upload/delete/copy 逻辑 |
| `server/app/main.py` | 移除 /uploads/ 端点 |
| `desktop/src/services/api.ts` | 更新 getMediaUrl() |
| `desktop/src/utils/markdown.ts` | 更新 resolveMediaUrl() |
