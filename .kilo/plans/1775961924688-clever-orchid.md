# Plan: Fix "内容无变化" and Enhance Change Summary

## Problem

1. **Bug**: Auto-save in `ArticleEditor.tsx` calls `updateArticleVersion` which updates the previous version's content to match the current edits. When the article is reopened and a new version is created, the comparison is against an already-updated version, resulting in "内容无变化" (no change).

2. **Enhancement**: The `_generate_change_summary` function in `articles.py` only shows generic line-level diffs. The user wants granular change detection:
   - Image additions/removals (`图片新增`/`图片删除`)
   - Table additions/removals (`表格新增`/`表格删除`)
   - General content changes
   - Deduplicated summary (same type of change shown only once)

## Changes

### 1. Fix auto-save version update bug (Client-side)

**File**: `desktop/src/pages/ArticleEditor.tsx` (line 246-248)

### 2. Enhance `_generate_change_summary` (Server-side)

**File**: `server/app/routes/articles.py` (lines 429-453)

Rewrite the function to detect specific content types:

- **Image detection**: Count `![alt](url)` patterns in old vs new content
  - Added images: present in new but not in old
  - Removed images: present in old but not in new
  - Summary: `图片新增 N 张` / `图片删除 N 张`

- **Table detection**: Count `|...|` header lines with `|---|` separators in old vs new
  - Added tables: present in new but not in old
  - Removed tables: present in old but not in new
  - Summary: `表格新增 N 个` / `表格删除 N 个`

- **General content changes**: Line-level diff (keep existing logic)
  - Summary: `内容新增 N 行` / `内容删除 N 行` (only if not already covered by specific types above, or always show alongside)

- **Title change**: Keep existing logic
  - Summary: `标题从「X」改为「Y」`

- Deduplication: Each type of change appears at most once

Proposed implementation:

```python
def _generate_change_summary(
    old_title: str | None,
    old_content: str | None,
    new_title: str,
    new_content: str,
) -> str:
    changes: list[str] = []

    # Title change
    if old_title is not None and old_title != new_title:
        changes.append(f"标题从「{old_title}」改为「{new_title}」")

    if old_content is not None and old_content != new_content:
        # Detect image changes
        old_imgs = set(re.findall(r'!\[[^\]]*\]\([^)]+\)', old_content))
        new_imgs = set(re.findall(r'!\[[^\]]*\]\([^)]+\)', new_content))
        added_imgs = new_imgs - old_imgs
        removed_imgs = old_imgs - new_imgs
        if added_imgs:
            changes.append(f"图片新增 {len(added_imgs)} 张")
        if removed_imgs:
            changes.append(f"图片删除 {len(removed_imgs)} 张")

        # Detect table changes
        old_tables = _count_tables(old_content)
        new_tables = _count_tables(new_content)
        if new_tables > old_tables:
            changes.append(f"表格新增 {new_tables - old_tables} 个")
        elif old_tables > new_tables:
            changes.append(f"表格删除 {old_tables - new_tables} 个")

        # Detect general line changes
        old_lines = old_content.splitlines(keepends=True)
        new_lines = new_content.splitlines(keepends=True)
        diff = list(difflib.unified_diff(old_lines, new_lines, n=0))
        added = sum(1 for l in diff if l.startswith("+") and not l.startswith("+++"))
        removed = sum(1 for l in diff if l.startswith("-") and not l.startswith("---"))
        if added or removed:
            parts: list[str] = []
            if added:
                parts.append(f"新增 {added} 行")
            if removed:
                parts.append(f"删除 {removed} 行")
            changes.append("，".join(parts))

    if not changes:
        return "内容无变化"
    return "；".join(changes)


def _count_tables(content: str) -> int:
    lines = content.split('\n')
    count = 0
    i = 0
    while i < len(lines):
        if '|' in lines[i] and i + 1 < len(lines) and re.match(r'^[\s|:-]+$', lines[i + 1]):
            count += 1
            i += 2
            while i < len(lines) and '|' in lines[i]:
                i += 1
        else:
            i += 1
    return count
```
