# 图文编辑自动保存间隔改为3秒

## 现状
- `desktop/src/pages/ArticleEditor.tsx:252` 已使用 `useAutoSave` hook，当前间隔为 **30000ms (30秒)**
- `desktop/src/utils/useAutoSave.ts` 是通用自动保存 hook，支持自定义间隔

## 修改
将 `ArticleEditor.tsx:252` 的 `useAutoSave` 调用中第三个参数从 `30000` 改为 `3000`：

```tsx
// 修改前
const { status: autoSaveStatus, lastSavedAtFormatted, markSaved } = useAutoSave(autoSaveFn, content + title, 30000);

// 修改后
const { status: autoSaveStatus, lastSavedAtFormatted, markSaved } = useAutoSave(autoSaveFn, content + title, 3000);
```

## 影响范围
- 仅修改1个文件1行代码
- `useAutoSave` hook 本身无需改动，它已经支持任意间隔参数
- 自动保存的防重复逻辑（`data === lastSavedRef.current` 时跳过）确保无意义保存不会执行
